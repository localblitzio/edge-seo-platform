What's happening on this page

  1. meta_rewrites — <title>, <meta name=description>, and og:title are all replaced server-side.
  2. canonical (custom strategy) — every existing <link rel=canonical> is stripped, and a new one pointing at https://lanterncrest.example/canonical-target is inserted.
  3. og:url / twitter:url sync — both are overwritten with the canonical URL.
  4. JSON-LD canonical sync — the existing <script type="application/ld+json"> has its top-level url and @id replaced with the canonical URL. The nested publisher.url is left alone (per §6.3 — only top-level).
  5. schema_injections × 3 — Article + FAQPage appended, BreadcrumbList prepended into <head>, each tagged with a data-edge-seo-rule marker for idempotence.
  6. element_removals — that red .demo-remove-me div is gone.
  7. content_injections — a green banner is inserted before the H1 heading, also marked for idempotence.
  8. link_rewrites — every external https://example.com/* link in the list below is rewritten to https://example.com/redirected/*.
  9. indexation — the original <meta name=robots> is replaced with the spec'd directives.
